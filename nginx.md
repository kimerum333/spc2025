# 엔진엑스
단순 웹서버가 아니라, 리버스프록시, 로드밸런싱, 캐싱 등의 기능이 있다.

## 마스터, 워커 프로세스
2개의 프로세스로 실행된다.
- 마스터 : 요청 수신 담당
- 워커(1개 이상) : 요청 처리 담당

## 설정
### conf 문법
nginx/nginx.conf
블럭으로 감싸진 블럭 디렉티브,
안 감싸진 심플 디렉티브로 구성
각각의 설정은 세미콜론으로 마무리;

### 설정파일의 분리
include / conf.d/ *.conf
