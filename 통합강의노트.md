# CORS
cors 는 기본적으로 브라우저의 보안 옵션.
1. 발생 조건
웹페이지를 준 서버와 다른 서버로부터 리소스를 추가적으로 요청할 시.
- =>브라우저가 CORS차단. 해커가 다른 서버로 사용자의 중요한 정보를 전송하려거나 할 수 있음.
2. 조건 상세
스키마가 다름(http, https), ip가 다름, 포트가 다름
